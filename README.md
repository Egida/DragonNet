# DragonNet
## Installation
### NextJS
```
$ cd app
$ npm install
$ npm run build
$ NODE_ENV=production node server.js
```
### RustLang & NodeJS ( In a bot )
```
1. Edit config.json to public ip and port or domain

$ cd tool
$ cargo build
$ cd ..
$ cd bot
$ npm install
$ node index.js
```
